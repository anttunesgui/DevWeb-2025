
INSTALAR O XAMPP <br>
  -Primeiro é necessário instalar o Xampp no seu computador para servir como servidor local. <br>
  -Nele voce deve intalar o php junto na hora da instalacao.
  -Ao abrir o programa ativar o Apache e o MySql.

CLONAR A PASTA DO TRABALHO <br>
  -Clone a pasta do trabalho no seu computador e salve no caminho do xampp, no caso a pasta HTDOCS.

CRIAR O BANCO DE DADOS <br>
  -Acessar o PgAdmin e criar um novo banco utilizando o script do arquivo sql/setup.sql.

NO ARQUIVO src/db.php <br>
  -Configurar a $connectionString com as chaves referente ao seu banco de dados.

ACESSAR NO NAVEGADOR <br>
  -Acessar o local onde esta o arquivo index.php <br>
  -Ex: localhost/Devweb-2025/TrabalhoFinal/public/index.php

SEGUINDO ESTES PASSOS DEVERA SER POSSIVEL EXECUTAR O TRABALHO
