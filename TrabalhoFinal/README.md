TRABALHO FINAL

INSTALAR O XAMPP
-Primeiro é necessário instalar o Xampp no ​​seu computador para servir como servidor local.
-Nele você deve instalar o php junto na hora da instalação. -Ao abrir o programa para ativar o Apache e o MySql.

CLONAR A PASTA DO TRABALHO
-Clone uma pasta do trabalho no seu computador e salve no caminho do xampp, no caso da pasta HTDOCS.

CRIAR O BANCO DE DADOS
-Acessar o PgAdmin e criar um novo banco utilizando o script do arquivo sql/setup.sql.

NO ARQUIVO src/db.php
-Configure a $connectionString com as chaves referentes ao seu banco de dados.

ACESSAR NO NAVEGADOR
-Acessar o local onde está o arquivo index.php
-Ex: localhost/Devweb-2025/TrabalhoFinal/public/index.php

SEGUINDO ESTES PASSOS DEVERA SER POSSIVEL EXECUTAR O TRABALHO
